Description du Projet

WillBank est une application bancaire complète développée dans le cadre d'un projet académique. Le projet repose sur une architecture microservices moderne composée de 8 services indépendants, garantissant une scalabilité et une maintenance optimale.

L'écosystème comprend Eureka Server (registre), API Gateway, Client Service, Compte Service, Transaction Service, Notification Service, Email Service et un Composite Service pour l'agrégation de données.

Architecture Technique

  • Backend : Java 17, Spring Boot 3.2, Spring Cloud 2023
  • Communication : OpenFeign (synchrone) & RabbitMQ (asynchrone)
  • Base de données : MySQL (une base par microservice)
  • Sécurité : Authentification JWT centralisée au niveau du Gateway
  • Frontend Web : Angular (Interface Administration)
  • Frontend Mobile : React Native / Expo (Interface Client)

Points Forts & Innovations

  • Event-Driven Architecture : Gestion des événements en temps réel via RabbitMQ (Topic & Direct)
  • Notifications Omnicanales : Emails HTML via SMTP Gmail et Push Notifications Firebase
  • Service Composite : Agrégation de données pour Dashboards et relevés unifiés
  • DevOps : Pipeline CI/CD configuré via GitLab CI
  • UI/UX : Deux interfaces distinctes pour admin et clients finaux
