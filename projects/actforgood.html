ActforGood

Angular React Native

Description

Plateforme sociale dédiée aux actions caritatives et au bénévolat. Connecte les organisations à but non lucratif avec des volontaires motivés pour créer un impact positif dans la communauté.

Objectifs

  • Faciliter l'engagement bénévole
  • Connecter les ONG avec des volontaires
  • Suivre l'impact des actions caritatives
  • Créer une communauté solidaire
  • Offrir une expérience cross-platform (web et mobile)

Expérience acquise

  • Développement d'applications hybrides
  • Architecture Angular pour le web
  • Développement mobile avec React Native
  • Intégration d'API REST
  • Gestion d'authentification utilisateur
  • Système de notifications push
