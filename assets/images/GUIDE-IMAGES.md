# Guide des Images pour votre Portfolio

## 📸 Images requises

### Photos personnelles
- **`profile.jpg`** - Photo de profil pour le header (400x400px recommandé)
- **`about-photo.jpg`** - Photo pour la section "À propos" (600x400px recommandé)

### Photos de projets
Placez vos captures d'écran dans le dossier `projects/` :
- **`projet-web-1.jpg`** - Capture de votre premier projet web
- **`projet-web-2.jpg`** - Capture de votre deuxième projet web
- **`design-mobile.jpg`** - Capture de votre design mobile
- **`design-web.jpg`** - Capture de votre redesign web
- **`photo-serie.jpg`** - Une de vos photos
- **`video-motion.jpg`** - Capture de votre vidéo motion design

## 🎨 Formats recommandés

### Photos de profil
- **Format** : JPG ou PNG
- **Taille** : 400x400px (carré)
- **Poids** : < 200KB

### Photos de projets
- **Format** : JPG ou PNG
- **Taille** : 600x400px (ratio 3:2)
- **Poids** : < 500KB chacune

## 🔧 Optimisation

Pour optimiser vos images :
1. **Redimensionnez** aux tailles recommandées
2. **Compressez** avec des outils comme TinyPNG
3. **Renommez** selon les noms indiqués ci-dessus

## 📁 Structure finale

```
assets/
├── images/
│   ├── profile.jpg
│   ├── about-photo.jpg
│   └── projects/
│       ├── projet-web-1.jpg
│       ├── projet-web-2.jpg
│       ├── design-mobile.jpg
│       ├── design-web.jpg
│       ├── photo-serie.jpg
│       └── video-motion.jpg
└── CV-Koubissack-Colette.pdf
```

## 🚀 Une fois les images ajoutées

Le portfolio utilisera automatiquement vos images. Si une image n'est pas trouvée, une image placeholder sera affichée.

**Conseil** : Utilisez des photos professionnelles et de bonne qualité pour faire la meilleure impression possible !
