# Dossier Assets

Placez ici vos fichiers :

## 📄 CV
- Copiez votre CV PDF ici
- Renommez-le : `CV-VotreNom.pdf`
- Exemple : `CV-Marie-Dupont.pdf`

## 🖼️ Images
- Photos de profil
- Captures d'écran de projets
- Logos et icônes

## 📁 Structure recommandée :
```
assets/
├── CV-VotreNom.pdf
├── images/
│   ├── profile.jpg
│   ├── projects/
│   │   ├── projet1.png
│   │   └── projet2.png
└── documents/
    └── certifications/
```

## 🔗 Utilisation dans le HTML :
```html
<a href="assets/CV-VotreNom.pdf" download>Télécharger CV</a>
<img src="assets/images/profile.jpg" alt="Photo de profil">
```
